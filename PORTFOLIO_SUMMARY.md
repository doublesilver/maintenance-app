## 🎤 면접 대비 답변

### Q: "이 프로젝트를 만든 동기는?"

> "더빌딩의 '바이브 코더' 채용 공고를 보고, 실제로 AI 도구를 활용해서 빠르게 프로젝트를 완성할 수 있다는 것을 증명하고 싶었습니다.
>
> '완벽한 설계보다 먼저 만들고 확인'이라는 철학이 공감되어, Claude Code를 활용해 2일 만에 MVP를 완성했고,
> 이후 비동기 처리, 실시간 알림 등 고급 기능을 추가하며 프로덕션 레벨까지 끌어올렸습니다."

### Q: "가장 어려웠던 부분은?"

> "동기 AI 처리로 인한 응답 지연 문제였습니다. 처음에는 요청당 2.5초가 걸렸는데,
>
> Celery + Redis로 비동기 작업 큐를 도입해서 0.1초로 단축했습니다.
>
> 사용자는 즉시 응답을 받고, AI 처리는 백그라운드에서 진행되도록 설계를 변경했습니다."

### Q: "AI 도구(Claude Code)를 어떻게 활용했나?"

> "단순 코드 생성을 넘어서 '페어 프로그래밍 파트너'로 활용했습니다.
>
> - 아키텍처 설계: '비동기 처리를 추가하려면 어떻게 해야 할까?' 질문 → 구조 제안
> - 코드 작성: 자연어로 기능 설명 → 전체 코드 생성
> - 디버깅: 오류 메시지 복사 → 원인 분석 및 해결책 제시
> - 문서화: 프로젝트 설명 → README/가이드 자동 작성
>
> 덕분에 혼자서도 4일 만에 풀스택 + 배포까지 완성할 수 있었습니다."

### Q: "프로덕션에 배포한 경험은?"

> "AWS EC2에 FastAPI를 배포하고, Nginx로 리버스 프록시를 설정했습니다.
>
> Systemd로 프로세스를 관리하고, CloudWatch로 로그를 모니터링합니다.
>
> GitHub Actions로 CI/CD 파이프라인을 구축해서, main 브랜치에 push하면 자동으로 테스트 → 배포가 진행됩니다.
>
> 프론트엔드는 Vercel에 배포했고, 파일은 S3에 저장합니다."
